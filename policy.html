Политика конфиденциальности

Политика конфиденциальности для мобильного приложения "MedixGO" 1. Общие положения 1.1. Настоящая Политика конфиденциальности (далее — Политика) регулирует обработку персональной информации (далее — Персональная информация), которая собирается и используется при использовании мобильного приложения "MedixGO" (далее — Приложение) и его функционала. 1.2. Персональная информация обрабатывается в строгом соответствии с законодательством Республики Казахстан о защите данных. 1.3. Настоящая Политика является неотъемлемой частью условий использования Приложения и может обновляться в одностороннем порядке для обеспечения соответствия законодательным требованиям или улучшения предоставляемых услуг. 2. Цели обработки персональной информации 2.1. Персональная информация собирается и обрабатывается для следующих целей: Авторизация и регистрация пользователей. Предоставление услуг через функционал Приложения (например, вызов медицинского персонала). Улучшение качества предоставляемых услуг. Обеспечение безопасности данных. Информирование пользователей о статусе их заказов и других аспектах использования Приложения. 3. Состав собираемой информации 3.1. Приложение может собирать следующую Персональную информацию: ФИО, номер телефона, адрес электронной почты, адрес проживания. Информация о заказах, выбранных услугах и предпочтениях пользователя. Данные геолокации (с согласия пользователя). Техническая информация: IP-адрес, тип устройства, версия операционной системы, данные о браузере. 3.2. Персональная информация предоставляется Пользователем добровольно в процессе регистрации и использования Приложения. 4. Хранение и защита данных 4.1. Все Персональные данные хранятся на серверах, соответствующих требованиям безопасности данных. 4.2. Правообладатель Приложения принимает все необходимые меры для защиты Персональной информации от несанкционированного доступа, изменения, раскрытия или уничтожения. 5. Передача данных третьим лицам 5.1. Персональная информация может быть передана третьим лицам только в следующих случаях: Для выполнения заказа (например, передача данных медицинскому персоналу). На основании требований законодательства. С согласия Пользователя. 5.2. Передача данных осуществляется в минимально необходимом объеме и в соответствии с положениями данной Политики. 6. Права пользователей 6.1. Пользователь имеет право: Запрашивать доступ к своей Персональной информации. Требовать исправления неточных или устаревших данных. Требовать удаления данных, если они более не нужны для целей обработки. Отозвать согласие на обработку своих данных. 6.2. Для реализации своих прав Пользователь может обратиться в службу поддержки по адресу: support@medixgo.kz. 7. Использование файлов cookie 7.1. Приложение может использовать файлы cookie для улучшения пользовательского опыта, хранения предпочтений и статистического анализа. 7.2. Пользователь может отключить использование файлов cookie через настройки своего устройства или приложения. 8. Обновление Политики 8.1. Настоящая Политика может быть изменена. Обновленная версия Политики будет опубликована в разделе «О приложении». 8.2. Пользователи будут уведомлены о существенных изменениях до их вступления в силу через уведомления в Приложении. 9.
